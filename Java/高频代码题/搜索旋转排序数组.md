每次循环，找到有序区间进行下一步查找  
有序区间必为 [0, mid] 或 [mid, len - 1] 之一  

```java
class Solution {
    public int search(int[] nums, int target) {
        int len = nums.length;
        int l = 0, r = len - 1;
        while (l <= r) {
            int mid = (l + r) / 2;
            if (nums[mid] == target)
                return mid;

            // 最右边有序
            if (nums[mid] < nums[len - 1]) {
                if (nums[mid] < target && nums[len - 1] >= target)
                    l = mid + 1;
                else
                    r = mid - 1;
            }

            // 最左边有序
            else {
                if (nums[0] <= target && nums[mid] > target)
                    r = mid - 1;
                else
                    l = mid + 1;
            }
        }
        return -1;
    }
}
```
